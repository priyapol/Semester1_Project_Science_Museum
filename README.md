# Science Community Museum Website

This is a responsive HTML and CSS website created for a Science Community Museum. The website provides information about the museum, opening hours, exhibitions, events, contact details, and ticket booking.

## Pages

- Home
- Opening Hours
- Explore
- Events
- Contact Us
- Book Now
- Coming Soon

## Features

- Responsive design for desktop, tablet, and mobile
- Navigation bar with hamburger menu
- Hero image section
- Explore cards for Kids and Teachers
- Events page layout
- Contact form design
- Book Now calendar/ticket layout
- Footer with quick links and social media icons

## Technologies Used

- HTML5
- CSS
- Font Awesome Icons
